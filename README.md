# COVID-19-India-State-wise-Dashboard-Power-BI

**Project Overview
**
This project presents an interactive Power BI dashboard analyzing COVID-19 data across Indian states. It provides a comprehensive view of active cases, cumulative cases, daily changes, and deaths, helping users understand the spread and impact of the pandemic.

The dashboard is designed with dynamic visuals, KPI cards, and map-based analysis to support data-driven insights.

**Objective**

To visualize COVID-19 data in an easy-to-understand format
To analyze state-wise performance and trends
To identify high-risk regions using interactive visuals
To enable quick and effective decision-making

**Dashboard Features - Key Metrics (KPIs)
**
Active Cases
Today Active Change
Cumulative Cases
Yesterday Change
Cumulative Deaths
Death During Day

**Visualizations**

Top 5 States by Cases – Identify most affected states
Top 5 States by Deaths – Analyze mortality distribution
Top 20 Active Cases by States – Track current hotspots
Map Visualization – State-wise geographic distribution

State Filter (Slicer) – Interactive analysis by state

**Tools & Technologies
**
Power BI Desktop
Power Query (ETL)
DAX (Data Analysis Expressions)

**Key Insights
**
Maharashtra and Kerala have the highest number of cases
Active cases highlight ongoing hotspots
Death trends help identify high-risk regions
Map visualization provides clear geographic insights
